# Hunt the Wumpus

## 📖 Description

A C++ implementation of the classic 1973 text-based adventure game by Gregory Yob.

---

## 🚀 Build

This project uses an **out-of-source build**. All generated build files and artifacts will reside in a separate `build` directory.

1. Create a build directory and navigate into it:
   ```bash
   mkdir build && cd build

2. Generate the build system files:
   ```bash
   cmake ..

3. Build the project:
   ```bash
   cmake --build .
